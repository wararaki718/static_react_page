# sample profile app

## start node server app (yarn)

```shell
yarn start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### build & serve (yarn)

```shell
yarn build
```

```shell
serve -s build
```

## build & serve (nginx)

```shell
docker-compose build
```

```shell
docker-compose up
```

Open [http://localhost:80](http://localhost:80) to view it in the browser.
